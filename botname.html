Bot change name

Bot Change Name adalah bot yang saya kembangkan khusus untuk komunitas SAMP (San Andreas Multiplayer), terutama dalam konteks bisnis Workshop (WS). Tujuan dari bot ini adalah untuk memberikan kebebasan kepada semua member di server Discord agar bisa mengubah nama mereka sendiri secara otomatis, tanpa perlu role admin atau moderator. Bot ini dibuat agar semua anggota baik pelanggan, mekanik, hingga GM—bisa mengatur nama mereka sendiri sesuai keperluan IC (In Character), tetapi tetap menjaga struktur jabatan atau posisi seperti CEO, Manager, Staff, Mekanik, dan sebagainya. Nama yang diganti tetap akan otomatis menggunakan format jabatan di depan nama, yang terkunci dan tidak bisa diubah oleh member. Misalnya: [CEO] Jonathan → diganti jadi Jonathan Nova, maka bot akan ubah otomatis menjadi: [CEO] Jonathan Nova Jadi, meskipun bebas ganti nama, jabatan tetap terjaga sesuai role di server.

  • Ganti nama otomatis lewat tombol di embed Discord tanpa perlu izin admin atau moderator.
  • Jabatan atau role-title seperti [CEO], [Mekanik], dll otomatis ditambahkan di depan nama dan tidak bisa diubah oleh user.
  • Format nama selalu konsisten dan menjaga kerapihan struktur jabatan di server.
  • Tampilan interaktif menggunakan embed dan button yang mudah digunakan oleh semua member.
  • Proses perubahan nama cepat, real-time, dan user-friendly.
  • Mengurangi beban admin dan moderator dalam menangani permintaan ganti nama.
  • Cocok untuk komunitas roleplay, bisnis IC, organisasi, dan server dengan sistem jabatan terstruktur.
